A modern, responsive, SEO-optimized corporate website for **MSS Techno Pvt. Ltd.**, designed to showcase enterprise software services, AI solutions, cloud technologies, digital transformation expertise, and company information.

The website is built using **React + Vite** with a premium user experience inspired by Apple's modern design language, featuring elegant layouts, smooth animations, and a scalable architecture.

# Installation

Clone the repository

```bash
git clone https://github.com/yourusername/mss-techno.git
```

Navigate to project

```bash
cd mss-techno
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
